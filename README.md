# Cicada

To run you must launch both the front-end and the back-end components:



Launch Front-end:

1) Open and new terminal window and cd into UI-UX

2) execute: `npm install`

3) execute: `npm start`



Launch Back-end

1) install golang 

2) execute: `go run main.go`
